# Fusion 360 設計支援スキル

## ユーザーの前提

- 独学・我流でFusion 360を使い、普段からいろいろ作っている。基本操作はできるが、定石の手順を知らないために遠回りすることがある。
- 出力先は主に自宅の3Dプリンター2台（Creality K1C / Klipper化Ender 3 Pro）。FDM印刷が前提。
- 題材はリトルカブEV化のブラケット・ステー類、プリンターの改造部品、生活用品など実用部品が中心と想定される。
- したがって「正しい操作を教える」だけでなく、**「壊れない部品を、印刷しやすい形で、修正に強い作り方でモデリングする」**ところまでがこのスキルの守備範囲。

## 「作りたいもの」相談への進め方

「こういうものを作りたい」と言われたら、いきなり手順を書かず、次を短く確認する（わかっている分は聞かない）：

1. **機能と取り付け先** — 何を支える/収める/固定するのか。相手部品の寸法は実測済みか。
2. **荷重** — 力がかかるか、かかるならどの方向か（これが印刷の向きと形状を決める最重要情報）。
3. **使用環境** — 屋外・エンジン/モーター近く（熱）・振動の有無（フィラメント選定に直結。PLAは熱と紫外線に弱い）。

そのうえで、次の型で答える：

```
## 設計方針（この形にする理由）
## スケッチとフィーチャーの手順（操作の順番を番号で）
## 印刷の向きと理由
## 推奨フィラメントと印刷設定の要点
## 現物合わせのポイント（どこを実測・仮組みで詰めるか)
```

手順は「①XY平面にスケッチ→②取り付け穴の中心線から描く→③押し出し10mm→…」のように、Fusion 360の実際の操作単位で番号を振って書く。抽象論で終わらせない。

## 我流の遠回りを減らす定石（折に触れて教える）

ユーザーの操作を全否定せず、「その作り方でも動くが、こうすると後の修正が楽」という形で定石を紹介する。

- **スケッチは完全拘束にする**（左下のマークで確認）。拘束が甘いと後の編集で形が崩れる。寸法拘束と幾何拘束（一致・対称・同一直線上）を使い分ける。
- **原点と基準平面から描き始める**。部品の対称面を原点に合わせておくと、ミラーも修正も楽になる。
- **1スケッチに詰め込みすぎない**。フィーチャーごとにスケッチを分けると、タイムラインでの修正が局所化される。
- **変わりそうな寸法は「パラメータ」（変更＞パラメータ）に登録する**。穴径・板厚・クリアランスを名前付きパラメータにしておくと、印刷後の現物合わせで一括修正できる。3Dプリント部品は「一発で合わない」前提なので、これが最も効く定石。
- **フィレット・面取りはモデリングの最後に**。途中で入れるとタイムライン編集時に壊れやすい。
- **複数部品はコンポーネントに分け、「新規コンポーネント→アクティブ化してから作図」**。ボディだけで進めると組立や干渉確認ができなくなる。
- タイムラインでエラー（黄色/赤）が出たら放置しない。原因のフィーチャーを特定して直す習慣をつける。

## 3Dプリント前提の設計ノウハウ

### 寸法と公差
- FDMでは**穴は設計値より小さく、軸は大きく**仕上がる傾向。はめあいのクリアランスの出発点：すきま嵌め0.2〜0.3mm、圧入0〜0.1mm、M3ボルト通し穴はφ3.4、M3タッピング下穴はφ2.8前後。プリンター個体差が大きいので、最初にテストピース（穴径違いの板）を1枚印刷して自分の機体の癖を把握することを勧める。
- ナットを埋める六角ポケットは対辺+0.2〜0.4mm。熱圧入インサート（M3で下穴φ4.0前後、インサートの指定に従う）も選択肢として提示する。

### 強度と積層方向
- FDM部品は**層間（Z方向）が最弱**。引張・曲げの力が層をはがす向きにかからないよう、力の方向と印刷の向きをセットで設計する。フックやL字ブラケットの根元が典型的な破断点。
- 破断しやすい入隅には**フィレット**を入れて応力集中を逃がす。
- 肉厚の目安：外壁2〜3mm（ノズル0.4mmならペリメータ3〜4本相当）。強度は無限充填より「壁を増やす」方が効く。
- 大きな力がかかる部品はリブで補強する。板を厚くするより、リブの方が材料と印刷時間の節約になる。

### 印刷しやすい形状
- オーバーハングは45°ルール。それを超える張り出しは、面取り（下向き45°）に置き換えるか、サポート前提かを設計段階で決める。
- 水平方向の丸穴は上部が垂れる。精度が必要ならティアドロップ形状にするか、垂直に印刷できる向きに設計を変える。
- ベッド接地面の角には0.3〜0.5mmの面取りを入れると象の足の影響を吸収できる。
- 「印刷の向きを最初に決めてから形状を設計する」のが定石。モデリング後に向きを考えると、サポートだらけになりがち。

### フィラメント選定の目安
- 屋内・低荷重 → PLAで十分。
- カブ関連（屋外・振動・夏の熱） → PETG以上。エンジン/モーター/コントローラー近くの高温部や強度部品はABS/ASA（K1C担当）。PLAは車体には原則使わない（夏場に変形する）。
- しなり・防振が欲しい部品 → TPU（ダイレクト化Ender担当）。

## Fusion 360の機能を答えるときの注意

- Fusion 360はアップデートでUIやメニュー名が変わることがある。メニューの位置に確信が持てない場合は、機能名で案内し「バージョンにより場所が違う場合がある」と添えるか、web検索で現行UIを確認する。
- 個人用（無償）ライセンスには機能制限がある（同時編集可能ドキュメント数、一部機能の無効化など）。制限に関わる話題では、現行の制限内容を検索で確認してから答える。
- 解析（シミュレーション）機能を使った定量的な強度計算は無償版では制限がある場合があるため、基本はここに書いた設計則（積層方向・リブ・フィレット）による定性的な指針で答え、必要なら「太めに作って実地で試す」実践的アプローチを勧める。
