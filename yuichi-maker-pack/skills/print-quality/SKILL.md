# 3Dプリンター印刷品質スキル

## ユーザーの機材（回答の前提として常に考慮する）

**1. Creality K1C**
- CoreXY機、エンクロージャー付き、高速印刷対応、純正でKlipperベースのファームウェア。
- 高温・エンクロージャーが必要なフィラメント（ABS/ASA）や炭素繊維入りはこちらが担当。
- 純正状態ではKlipperの設定変更に制限がある（rootを取っていなければprinter.cfgの自由編集は不可）。設定変更を提案する前に、root化済みかどうかを確認する。

**2. Ender 3 Pro（大幅改造済み）**
- 基板：BIGTREETECH SKR Mini E3 **V3.0** に換装済み（2026年7月、V1.2から更新）。ステッパードライバはTMC2209オンボードで、**UARTバスは共有構成**（各ドライバをアドレスで区別）。printer.cfg関連の助言はKlipper公式のgeneric-bigtreetech-skr-mini-e3-v3.0.cfgを基準にし、V1.2/V2.0向けのピン定義を流用しない。
- BLTouchは**sensor_pin: PC14 / control_pin: PA1**で稼働中（換装時に本体も交換済み、Z-offset調整済み）。
- デュアルZ軸は**Z2をZ1と並列接続で運用**（printer.cfgのstepper_z1セクションは削除済み）。デュアルZの助言では独立2ドライバ構成を前提にしない。
- ダイレクトエクストルーダー化、Klipper化済み。日本語表示の文字化け対策済み。
- ヒートクリープ起因のフィラメントジャム歴あり。ABS製ヘッドカバーに換気穴を追加して対処中のため、ジャム再発の相談ではまず冷却(ヒートシンク周りの通風)を疑う。
- 改造はユーザー自身が実装しており、ハードの理解はある。ただしKlipperの設定ファイルはAI任せで書いた経緯があるため、printer.cfgの中身は本人も完全には把握していない可能性がある。設定変更を提案するときは「該当セクションの現在の記述を貼ってもらう」ことから始めると安全。
- ベッドスリンガー機なので、高速化はY軸の慣性が品質のボトルネックになる。

**スライサー**：Creality PrintとCuraを併用。設定名はどちらのスライサーの話かを明示する（同じ機能でも名称が違う。例：Curaの「コースティング」、プレッシャーアドバンスとの併用は非推奨、など）。

## 回答の基本方針

1. **症状→原因の切り分けを先に、対策は一度に一つ** — 印刷不良は複数原因が絡むが、一度に複数の設定を変えると何が効いたかわからなくなる。可能性の高い原因から順に、1変更→1テスト印刷のサイクルを提案する。
2. **どちらのプリンターの話かを最初に確認する** — 同じ症状でも、CoreXYのK1CとベッドスリンガーのEnder 3では原因の優先順位が違う（例：ゴースティングはEnderならY軸のベッド慣性、K1Cならベルトテンションを先に疑う）。
3. **機械→ファームウェア→スライサーの順で疑う** — ベルトの緩み、偏心ナットのガタ、ノズルの摩耗などの機械的問題は、設定をいくらいじっても直らない。まず機械的な健全性を確認してから設定の話に進む。
4. **printer.cfgを変更する提案では安全を最優先** — 変更前のバックアップ、`SAVE_CONFIG`と手動編集の使い分け、変更後は必ずファームウェア再起動、Z関連の変更後は最初の一層を手を離さず監視する、といった手順を必ず添える。サーマルランナウェイ保護に関わる設定は絶対に無効化しない。

## キャリブレーションの標準手順（順番が重要）

「全体的にきれいにしたい」という相談には、この順番での実施を案内する。順番を飛ばすと後工程の調整が無意味になる。

1. **機械の点検** — ベルトテンション、ローラーのガタ、各部のネジ、ノズルの状態
2. **Z オフセットとベッドメッシュ** — 3D Touch / K1Cの自動キャリブレーション。一層目がすべての土台
3. **rotation_distance（押出量の較正）** — Klipperではe-stepsではなくrotation_distance。100mm押出テストで実測
4. **フローの調整** — スライサー側。一層壁のキューブなどで実測
5. **温度タワー** — フィラメント銘柄ごとに最適温度を確認
6. **リトラクションテスト** — ダイレクト機は短め（0.5〜1.5mm程度から）が出発点
7. **プレッシャーアドバンス** — Klipperのチューニングタワーで較正
8. **インプットシェーパー** — ゴースティング対策。加速度センサーがなければ手動のリングテストでも可

## 症状別の初動（よくあるもの）

- **糸引き** → 温度が高すぎ or リトラクション不足。まず温度タワー、次にリトラクションテスト。
- **ゴースティング（残像状の縞）** → 機械の剛性・ベルト → 加速度を下げる → インプットシェーパー。
- **一層目がつかない/潰れすぎ** → Zオフセット再調整とベッドメッシュ。ベッドの清掃（中性洗剤 or IPA）も定番。
- **象の足** → 一層目の潰しすぎ or ベッド温度過多。スライサーの「初期レイヤー水平拡張（マイナス値）」で補正も可。
- **押出不足（スカスカ）** → ノズル詰まり・フィラメント湿気 → rotation_distance → フロー、の順。
- **層ズレ** → ベルトテンション、モータードライバ電流（Klipperの`run_current`）、速度・加速度の出しすぎ。
- **ブロブ・ジット** → プレッシャーアドバンス未調整、シーム設定、コースティングとPAの併用（併用は避ける）。

## フィラメント別の割り当ての目安

- **PLA** → どちらでも。K1Cで印刷する場合はエンクロージャーの熱こもりに注意（扉/天板を開ける）。
- **PETG** → 糸引きが出やすいので温度とリトラクションの詰めが重要。ベッドへの食いつきすぎに注意。
- **TPU** → ダイレクト化したEnder 3が適任。速度を大きく落とし（20〜30mm/s目安）、リトラクション最小限。
- **ABS/ASA** → エンクロージャーのあるK1C一択。換気にも言及する。

## トラブル相談のときに聞くこと

情報が足りない場合、次をまとめて確認する（質問攻めにせず一度で）：(1) どちらのプリンターか、(2) フィラメントの種類と銘柄、(3) 症状の写真、(4) 最近変えたこと（設定・部品・フィラメント）。Klipperの設定が関わりそうなら printer.cfg の該当セクションを貼ってもらう。
