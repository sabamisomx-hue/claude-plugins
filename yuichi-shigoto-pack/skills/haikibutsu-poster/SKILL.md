# 病院廃棄物分別・啓発ポスター作成スキル

## 前提

- ユーザー（ゆういちさん）は**660床病院の特別管理産業廃棄物担当者**。感染性廃棄物・一般廃棄物などの分別の啓発、委員会報告、JWNet（電子マニフェスト）運用を担っている。
- ポスター・分別表の読み手は**看護師・医師などの医療職**。忙しい現場で「立ち止まらず一瞬で判断できる」ことが最重要。読ませる掲示物ではなく、見て分かる掲示物を作る。
- 私が作れるのは、標語・コピーの案出しと、**SVG/HTMLベースのポスター・分別表デザイン**（文字・図形・色・アイコン構成。A4/A3印刷向け）。写真やイラスト調の画像生成はできないので、写真を使いたい場合は写真の配置枠を設けたレイアウトを作り、現場写真をはめ込んでもらう形を提案する。

## ルールの根拠と、勝手に決めてはいけないこと

- 感染性廃棄物の判断基準は環境省「**廃棄物処理法に基づく感染性廃棄物処理マニュアル**」が公式の拠り所。判断フロー（①廃棄物の形状 → ②排出場所 → ③感染症の種類）の考え方に沿う。マニュアルの内容を具体的に引用・記載するポスターを作る場合は、web検索で現行版を確認する。
- **バイオハザードマークの色分け**（性状に応じた区分）：
  - **赤**：液状・泥状のもの（血液・体液など)
  - **橙**：固形状のもの（血液等が付着したガーゼ、手袋など）
  - **黄**：鋭利なもの（注射針、メス、アンプルなど。破損ガラスも鋭利物扱い）
- ただし、**容器の運用・分別区分の細部は施設ごとに異なる**（契約する処理業者、院内規程、非感染性の扱い、プラスチックの分別など）。ポスターに載せる具体的な分別区分・容器の種類・色は、必ずユーザーの院内ルールを確認してから記載する。一般論で埋めて「それらしいが院内ルールと違うポスター」を作ることが最悪の失敗。院内ルールが未整理の部分が見つかったら、それ自体を委員会への論点として提起することを提案する。

## ポスターづくりの原則（医療現場向け）

1. **1ポスター1メッセージ**：「針は黄色容器へ」「ガーゼと針を一緒にしない」など、伝えることを一つに絞る。総合分別表とメッセージポスターは別物として作る。
2. **3秒で伝わる構成**：大きな見出し（何をするか）→ 視覚要素（色・アイコン・○×）→ 補足は最小限。文章を読ませない。
3. **色は現物と一致させる**：ポスター上の色分けは、院内で実際に使っている容器・マークの色と必ず一致させる。ここがズレると啓発が逆効果になる。
4. **○×の対比が最強**：「正しい例○／ありがちな間違い×」の対比レイアウトは、規則の羅列より圧倒的に伝わる。よくある分別ミス（針を橙容器へ、一般ごみへの感染物混入、逆に何でも感染容器に入れてコスト増）を×側の題材にする。
5. **理由を一行添える**：「なぜダメか」（針刺し事故・処理費・法令）を小さく一行入れると、単なる命令より遵守率が上がる。ただし主役にはしない。
6. **貼る場所から逆算**：ナースステーション、処置室、汚物処理室、容器の直上など、掲示場所を確認してサイズと文字の大きさを決める（容器直上なら至近距離・A4、廊下ならA3以上で遠目に読める文字）。
7. **多職種・多国籍への配慮**：委託清掃スタッフ等も読む掲示なら、ふりがな・平易な語・ピクトグラム優先を提案する。

## 作れる成果物

- **メッセージポスター**（A4/A3縦横）：SVGで作成し、PDF化して印刷用に出力。院内で色校正しやすいよう、使用色は単純明快に。
- **分別早見表**：品目→容器の対応表。壁貼り用の表形式（SVG/PDF）と、改訂管理しやすいExcel原本の両方を作れる。品目リストはユーザーから提供してもらう（勝手に品目を創作しない）。
- **判断フローチャート**：「迷ったらこれ」の分岐図（これは感染性？→形状は？→どの容器？）。visualizerでプレビューを見せながら詰められる。
- **委員会向け資料**：分別ミスの発生状況報告、啓発計画などはgyomu-bunshoスキルの型で作成し、掲示物の縮小版を添付資料にする連携も可能。
- **教育用ミニテスト・配布資料**：新人看護師向けの分別クイズなどはshinjin-kenshuスキルと連携。

## 制作の進め方

1. 伝えたいメッセージ（または直したい分別ミス）と、掲示場所・サイズを確認する。
2. 院内の分別区分・容器の色運用を確認する（既に聞いた内容は再確認しない。初回に確認したらこのスキルへの追記を提案し、以後の制作を速くする）。
3. 標語・コピー案を3〜5案出す（直球型・注意喚起型・理由提示型など方向性を変えて）。
4. 選ばれた案でSVGレイアウトを作成し、visualizerでプレビュー → 修正 → 印刷用ファイル（PDF/SVG）を出力する。
5. 文言の最終確認をユーザーに促す（法令・院内規程に関わる表現の最終責任者はユーザー）。

## 表現上の注意

- 恐怖を煽りすぎる表現（生々しい事故描写）は避け、行動指示を主役にする。針刺し事故等に触れる場合も事実ベースで簡潔に。
- 特定の職種を責める書き方（「看護師の分別ミスが多発」等）を掲示物にしない。行動に焦点を当てる（「針とガーゼは別容器へ」）。
- 法令名・マニュアル名を記載する場合は正式名称を検索で確認する。
