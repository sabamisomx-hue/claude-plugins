# 新人研修教材作成スキル

## このスキルの目的

ユーザー（ゆういちさん）が担当する新入社員研修（事務・総合職を含む全般対象）の教材づくりを、企画から成果物まで一貫して支援する。単に「それらしい資料」を出すのではなく、**研修が終わったとき新人が何をできるようになっているか（到達目標）から逆算して**教材を設計する。

## 教材作成の前に確認すること

会話から読み取れない場合、最初にまとめて確認する：

1. **対象と人数** — 新卒か中途か、人数、配属予定（現場系/事務系が混在するか）。
2. **時間枠** — 持ち時間（例：90分×1コマ、半日、3日間）。
3. **到達目標** — 研修後に「できるようになっていてほしいこと」。曖昧なら「終了時に新人が〇〇できる」の形に言語化するのを手伝う。
4. **形式** — 講義中心か、ワーク・ロールプレイを入れるか、テストの有無。
5. **会社固有の情報** — 社内ルール・組織・沿革などが必要な教材では、勝手に創作せず【会社名】【部署名】のようなプレースホルダーにするか、情報提供を依頼する。**会社固有の事実をでっち上げないことを徹底する。**

## 設計の定石（教材の質を決める部分）

- **逆向き設計**：到達目標 → それを測る確認方法（テスト・ワーク）→ 必要な講義内容、の順で組む。「話したいこと」からではなく「できるようになってほしいこと」から作る。
- **集中力の単位は15〜20分**：講義はこの単位で区切り、間にワーク・ペア討議・質疑を挟む構成を提案する。90分ノンストップの一方通行スライドは作らない。
- **具体例とNG例をセットで**：マナー系は特に「正しい例」だけでなく「ありがちな失敗例→なぜまずいか→正しい形」の流れが記憶に残る。
- **新人の前提知識ゼロで書く**：社内用語・業界用語には必ず注釈。「報連相」のような定番語も、意味と使いどころを具体的な場面で示す。
- **演習は現実の場面設定で**：電話応対なら実際にかかってきそうな用件、メールなら実際に送りそうな相手を設定したロールプレイ・添削課題にする。

## 定番コンテンツの引き出し

依頼に応じて以下の領域の教材を作成できる。内容は一般的なビジネス慣行に基づき、会社独自ルールがある場合は上書きしてもらう前提で作る。

- **ビジネスマナー**：挨拶・身だしなみ、敬語（尊敬語/謙譲語/丁寧語の使い分けと頻出の誤用）、電話応対（受け方・取り次ぎ・かけ方）、名刺交換、来客対応・席次、ビジネスメールの型
- **仕事の進め方**：報連相の実践（悪い報告ほど早く、事実と意見の区別）、メモの取り方、指示の受け方（復唱・不明点の確認）、優先順位とスケジュール管理
- **コンプライアンス・情報管理**：SNSの注意、機密情報の扱い、ハラスメントの基礎知識
- **安全衛生の基礎**：事務職含む全員向けの一般的な内容（整理整頓、緊急時対応）。現場系の専門的な安全教育が必要な場合は内容が変わるため、対象を確認する。

## 成果物の形式

用途に応じて適切なファイル形式で作成する（各ドキュメントスキルを併用）：

- **カリキュラム表・時間割** → Excel（時刻・項目・形式・担当・準備物の列構成）
- **講義スライド** → PowerPoint。1スライド1メッセージ、文字を詰め込まない。講師が口頭で補足する前提の作り。
- **配布資料・ワークシート** → Word。書き込み欄のあるワーク形式を活用。
- **確認テスト** → 問題と解答・解説を分離して作成。到達目標と対応づけ、「講義で扱っていないことを出題しない」を守る。
- **講師用進行台本** → 時間配分、話す要点、ワークの指示文、想定質問と回答例まで含める。ユーザー以外が講師を務めても回る粒度を目指す。

## 作成時の注意

- 研修資料はそのまま社内で使われる成果物なので、誤字・敬語の誤りは信頼を損なう。マナーを教える資料自体の言葉遣いには特に注意する。
- 世代論やステレオタイプ（「最近の若者は〜」）を教材に持ち込まない。行動ベースで書く。
- 労働法・就業規則に関わる内容（残業、休暇等）は会社の規程が優先。一般論を書く場合は「就業規則を確認」と必ず添える。
- 実施後の改善まで見据え、振り返りアンケート（理解度・難易度・要望）の設計も提案できる。
